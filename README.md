# e-plantShopping

## Project Overview

**e-plantShopping** is a modern e-commerce web application designed for purchasing houseplants and gardening products online. Built with React and Redux Toolkit, this application provides users with an intuitive shopping experience featuring a dynamic product catalog, shopping cart functionality, and seamless state management.

### Key Features

- **Product Catalog**: Browse a curated selection of houseplants with detailed descriptions and pricing
- **Shopping Cart**: Add, remove, and manage items in your cart with real-time updates
- **Redux State Management**: Efficient state handling using Redux Toolkit for cart operations
- **Responsive Design**: Modern, mobile-friendly interface built with React
- **Fast Performance**: Powered by Vite for optimal development and production builds

### Technologies Used

- React
- Redux Toolkit
- Vite
- CSS3
- JavaScript (ES6+)

This project demonstrates modern web development practices and provides a foundation for building scalable e-commerce applications.
